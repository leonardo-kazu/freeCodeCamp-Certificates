Introduction

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python's elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

The Python interpreter and the extensive standard library are freely available in source or binary form for all major platforms from the Python web site, https://www.python.org/, and may be freely distributed. The same site also contains distributions of and pointers to many free third party Python modules, programs and tools, and additional documentation.

Another important factor of Python is that in indentation is is key, since Python is a dynamic language, it's been made so that scopes are decided trough indentation.

Variables

Variables are symbolic names for values in your application. The names of variables, identifiers, conform to some rules.

A Python identifiers cannot be a protected keyword of the Python language, also, Python is case sensitive, this means that the variable: Test and test are two different variables.

Declaring variables

In Python there's no need to declare a variable with a certain data type, this's because it automatically determines the type in accordance of the value assigned.

For example: 

x = 42 #Declaring an integer
y = 1.2 #Declaring a float
name = "howdy" #Declaring a string

Variable scope

All variables in Python are locally scope, in case the local of the scope is the global document, the variable is a global scope variable, in case it's declared inside a fuction, the variable will only be acessible inside that function.

Constants

By default Python does not have constants, so to resolve this probleam, it's widely addepted that constants be declared in all caps, with underscores (_) for spaces. For example:

THIS_IS_MY_CONSTANT = 3.14

Beware that, you will still be available to change the variable value. This is only an aesthetic change so it becomes easier to understand.

if...else statement

Use the if statement to execute a statement if a logical condition is true. Use the optional else clause to execute a statement if the condition is false. An if statement looks as follows:

if (condition):
    statement_1
else:
    statement_2

condition can be any expression that evaluates to true or false, if the condition evaluates to true, statement_1 will be executed, otherwise statement_2 will be executed.

In case that there's a need for more than one condition to be tested in sequence, the following code could be used:

if (condition_1):
    statement_1
elif (condition_2):
    statement_2
elif (condition_n):
    statement_3
else:
    statement_last

In the case multiples conditions are true, only the first to evaluated to true will be executed, and the others will be discarted, if none of the conditions are true, then the else statement will be executed

while statement

The while statement is used for repeated execution (loop) as long as a condition is true:

while (condition):
    statement

This repeatedly tests the condition and, if it is true, executes the statement. If the condition is false, the loop is terminated.

Function declarations

A function definition consists of:

  • The keyword: def
  • The name of the function
  • A list of the arguments of the function, in parentheses and separathed by commas.
  • The statement that will run in the correct indentation.

As shown in the following code:

def function_name(arg):
    statement

The return keyword, specifies the value returned by the function (if the function returns something)
